# Push Button and LED Interfacing using STM32

## 📌 Objective

To interface a push button as digital input with an LED as digital output.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- Push Button
- LED

---

## ⚙️ Working Principle

The push button is configured as GPIO input. Based on the button state, the LED is turned ON or OFF.

---

## 📷 Output

LED changes state when the push button is pressed.

---

## 🧠 Concepts Learned

- GPIO input configuration
- Pull-up/pull-down resistors
- Sensor-actuator interfacing
