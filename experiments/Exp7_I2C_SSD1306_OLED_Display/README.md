# SSD1306 OLED Display Interfacing using I2C

## 📌 Objective

To interface an SSD1306 OLED display using I2C and display sensor or text data.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- SSD1306 OLED Display

---

## ⚙️ Working Principle

The OLED display communicates with STM32 through I2C. Text and sensor values are displayed in real-time.

---

## 📷 Output

Text and sensor readings are successfully displayed on the OLED screen.

---

## 🧠 Concepts Learned

- OLED display interfacing
- I2C communication
- Real-time embedded display systems
