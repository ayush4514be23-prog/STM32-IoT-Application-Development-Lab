# Multi-Sensor Telemetry System using STM32

## 📌 Objective

To aggregate data from multiple onboard sensors and structure it into a telemetry frame.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- Multiple onboard sensors

---

## ⚙️ Working Principle

Sensor data from multiple peripherals is collected, processed, and formatted into a telemetry packet.

---

## 📷 Output

Structured telemetry data is generated successfully.

---

## 🧠 Concepts Learned

- Sensor data acquisition
- Data aggregation
- Embedded telemetry systems
