# PWM Servo Motor Control using STM32

## 📌 Objective

To generate PWM signals using timers and control the angular position of a servo motor.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- Servo Motor

---

## ⚙️ Working Principle

PWM signals generated using STM32 timers control the rotation angle of the servo motor.

---

## 📷 Output

Servo motor rotates to the desired angular positions.

---

## 🧠 Concepts Learned

- PWM generation
- Timer peripherals
- Servo motor control
