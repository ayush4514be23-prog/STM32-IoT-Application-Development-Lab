# BLE Sensor Data Broadcasting using STM32

## 📌 Objective

To configure BLE communication and broadcast sensor data using BLE beacon packets.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- BLE Module

---

## ⚙️ Working Principle

Sensor data is transmitted wirelessly using BLE advertisement packets.

---

## 📷 Output

Sensor data is successfully broadcast over BLE.

---

## 🧠 Concepts Learned

- BLE communication
- Wireless embedded systems
- IoT connectivity
