# HTS221 Temperature and Humidity Sensor using I2C

## 📌 Objective

To interface the HTS221 sensor using I2C and measure temperature and humidity data.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- HTS221 Sensor

---

## ⚙️ Working Principle

The HTS221 sensor communicates with STM32 using the I2C protocol. Temperature and humidity values are acquired and processed.

---

## 📷 Output

Real-time temperature and humidity values are obtained from the sensor.

---

## 🧠 Concepts Learned

- I2C communication
- Sensor interfacing
- Environmental monitoring systems
