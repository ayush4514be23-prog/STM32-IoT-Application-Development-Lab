# WiFi HTTP Data Transmission using STM32

## 📌 Objective

To establish WiFi connectivity and transmit sensor data to a remote server using HTTP requests.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- WiFi Module

---

## ⚙️ Working Principle

STM32 connects to a WiFi network and sends sensor data to a remote server using HTTP communication.

---

## 📷 Output

Sensor data is transmitted successfully over WiFi.

---

## 🧠 Concepts Learned

- WiFi networking
- HTTP communication
- Embedded IoT systems
