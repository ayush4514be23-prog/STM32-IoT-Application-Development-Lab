# LPS22HB Pressure Sensor using I2C

## 📌 Objective

To interface the LPS22HB sensor using I2C and acquire atmospheric pressure data.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- LPS22HB Pressure Sensor

---

## ⚙️ Working Principle

The STM32 communicates with the LPS22HB pressure sensor through the I2C protocol and reads pressure measurements.

---

## 📷 Output

Atmospheric pressure data is successfully displayed and monitored.

---

## 🧠 Concepts Learned

- Pressure sensor interfacing
- I2C communication
- Embedded environmental sensing
