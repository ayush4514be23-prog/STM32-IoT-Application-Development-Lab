# LSM6DSL Accelerometer Interfacing using SPI

## 📌 Objective

To configure SPI communication and read motion data from the LSM6DSL accelerometer sensor.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- LSM6DSL Motion Sensor

---

## ⚙️ Working Principle

The STM32 microcontroller communicates with the LSM6DSL sensor using the SPI protocol to acquire acceleration data.

---

## 📷 Output

Motion and acceleration values are successfully acquired from the sensor.

---

## 🧠 Concepts Learned

- SPI communication
- Motion sensor interfacing
- Embedded sensor systems
