# USART Serial Communication using STM32

## 📌 Objective

To configure the USART peripheral for asynchronous serial communication.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- USB UART Interface

---

## ⚙️ Working Principle

USART peripheral is initialized for serial transmission of text data to a terminal application.

---

## 📷 Output

Text messages are displayed on the serial terminal.

---

## 🧠 Concepts Learned

- USART configuration
- Serial communication
- STM32 peripheral initialization
