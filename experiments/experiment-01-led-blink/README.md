# Experiment 01 – GPIO LED Blinking

## Aim

To configure a GPIO pin as digital output and implement LED blinking.

## Hardware

STM32L475 IoT Discovery Board

## Software

STM32CubeMX
STM32CubeIDE

## Description

This experiment demonstrates basic GPIO configuration in STM32.
The LED connected to PB14 is toggled with a delay to produce a blinking effect.
