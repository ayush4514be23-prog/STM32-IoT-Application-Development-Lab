# STM32 IoT Experiments

This folder contains all STM32 IoT laboratory experiments developed using the STM32L475 IoT Discovery Board.

## Experiments Included

- GPIO LED Blinking
- Button and LED Interfacing
- USART Communication
- I2C Sensor Interfacing
- SPI Communication
- OLED Display
- PWM Servo Control
- BLE Communication
- WiFi HTTP Communication
