# GPIO LED Blinking using STM32

## 📌 Objective

To configure a GPIO pin as digital output and implement LED blinking using STM32.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- Onboard LED

---

## ⚙️ Working Principle

The GPIO pin connected to the onboard LED is configured as output. The LED state is toggled periodically to create a blinking effect.

---

## 📂 Project Structure

```text
Core/
Drivers/
Docs/
```

---

## 📷 Output

LED blinks continuously at fixed intervals.
