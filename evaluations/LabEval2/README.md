# Lab Evaluation 2 - Sensor Interfacing

## 📌 Objective

To interface onboard sensors with STM32 and acquire environmental data.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- HTS221 Sensor
- LPS22HB Sensor

---

## ⚙️ Features

- Sensor Data Acquisition
- I2C Communication
- Environmental Monitoring

---

## 📷 Output

Temperature, humidity, and pressure data acquired successfully.

---

## 🧠 Concepts Learned

- Sensor interfacing
- I2C protocol
- Embedded sensing systems
