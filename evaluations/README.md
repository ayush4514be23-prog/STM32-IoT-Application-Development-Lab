# STM32 IoT Laboratory Evaluations

This directory contains laboratory evaluation projects developed using the STM32L475 IoT Discovery Board as part of the IoT Application Development course.

## 📂 Evaluation List

| Evaluation | Description |
|---|---|
| LabEval1 | GPIO and USART Evaluation |
| LabEval2 | Sensor Interfacing Evaluation |
| LabEval3 | Servo Motor and OLED Display System |
| LabEval4 | MQTT, WiFi and USART Communication System |

---

## 🛠 Hardware Platform

- STM32L475 IoT Discovery Board
- SSD1306 OLED Display
- Servo Motor
- WiFi Module
- USART Terminal
- Environmental Sensors

---

## 💻 Software Tools

- STM32CubeIDE
- STM32CubeMX
- Embedded C
- STM32 HAL Drivers
- MQTT Protocol
- Serial Terminal Applications

---

## 📌 Features Covered

- GPIO Programming
- USART Communication
- I2C Communication
- SPI Communication
- PWM Servo Control
- OLED Display Interfacing
- MQTT Communication
- WiFi Networking
- Embedded IoT Systems
