# Lab Evaluation 3 - Servo Motor and OLED Display System

## 📌 Objective

To control a servo motor using PWM signals and display data on an OLED display.

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- Servo Motor
- SSD1306 OLED Display

---

## ⚙️ Features

- PWM Servo Control
- OLED Display Output
- Real-Time Embedded Display System

---

## 📷 Output

Servo movement and OLED display functionality verified successfully.

---

## 🧠 Concepts Learned

- PWM generation
- OLED interfacing
- Real-time embedded systems
