## 📌 Overview

This repository contains all laboratory experiments and projects developed for the **IoT Application Development (23EC362)** course using the **STM32L475 IoT Discovery Board**.

The repository demonstrates practical implementation of:

- GPIO Programming
- UART / USART Communication
- I2C and SPI Protocols
- PWM Motor and LED Control
- BLE Communication
- WiFi Networking
- MQTT Protocol
- Embedded Web Server Development
- Sensor Interfacing
- Real-Time Embedded Systems

---

## 🛠 Hardware Used

- STM32L475 IoT Discovery Board
- ESP8266 / ESP32
- Ultrasonic Sensors
- DHT22 Sensor
- OLED Display

- LEDs and Push Buttons
